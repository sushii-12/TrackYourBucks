# TrackYourBucks 💸

A clean, modern expense tracker built with Flask and SQLite.

## Features
- Add income and expenses
- Monthly summaries
- Average daily spend
- Export transactions as CSV
- Dark neon UI with glassmorphism

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
- Bootstrap

## How to Run Locally
```bash
git clone https://github.com/sushii-12/TrackYourBucks
cd trackyourbucks
python app.py
